Documentación de la API

Primeros Pasos

La API de Precios TUR proporciona acceso a los precios regulados actuales e históricos del gas en España. La API es publica y no requiere ningun tipo de authentication.

Hostname

Para acceder a la API, simplemente utiliza el hostname.

https://test.com

Obtener Precios Actuales

GET/api/v1/prices/current

Devuelve los precios TUR actuales para todas las categorías

Example Response

{
  "data": [
    {
      "category": "TUR1",
      "variableRate": 0.05567,
      "fixedRate": 4.35,
      "date": "2024-01-01"
    }
  ]
}

Obtener Historial de Precios

GET/api/v1/prices/history

Devuelve el historial de precios TUR, opcionalmente filtrado por categoría

Parameters

category
string - Categoría TUR (TUR1, TUR2, o TUR3)

Example Response

{
  "data": [
    {
      "category": "TUR1",
      "prices": [
        {
          "date": "2024-01-01",
          "variableRate": 0.05567,
          "fixedRate": 4.35
        }
      ]
    }
  ]
}
